Streaming TTS Alphanumeric Comparison

Compare TTS providers: ElevenLabs, Deepgram, Cartesia, and Rime

How to Use This Demo

1
Choose a challenge: Select a pre-set challenge from the dropdown, enter custom text, or click "Generate Random Challenge" to create a new test case.
2
Configure providers: Use the toggle checkboxes to enable/disable optional providers (Cartesia and Rime). Choose different voices for each enabled provider using the voice dropdowns. Changing a voice will reset that provider's audio.
3
Start the test: Click "Start TTS Test" to generate audio from all enabled providers simultaneously. The Time-to-First-Audio (TTFA) metric will be displayed for each provider.
4
Play & compare: Once buffering is complete, click "Play" buttons to hear each provider's pronunciation. Use "Download Audio" to save individual audio files.
